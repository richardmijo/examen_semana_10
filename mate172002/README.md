El propósito del proyecto es profundizar en el uso de estructuras de datos avanzadas para la gestión efectiva de tareas y proyectos. A medida que los proyectos se vuelven más complejos y dinámicos, es fundamental utilizar soluciones que permitan una mejor organización, control y ejecución. Las estructuras de datos seleccionadas —Listas Enlazadas, Listas Circulares, Listas Doblemente Enlazadas y Recursividad— son herramientas poderosas que facilitan la manipulación y procesamiento de datos en contextos donde la eficiencia y la flexibilidad son esenciales.

Objetivos del Proyecto:
Implementar Listas Enlazadas para la gestión de proyectos y la asignación de tareas dinámicas, mejorando la flexibilidad en el manejo de datos relacionados.

Explorar Listas Circulares para manejar tareas recurrentes o cíclicas, permitiendo un flujo continuo y la reutilización de recursos sin necesidad de reiniciar desde el inicio.

Aplicar Listas Doblemente Enlazadas para la planificación bidireccional, facilitando la navegación a través de tareas y proyectos complejos que requieren seguimiento en ambas direcciones.

Utilizar Recursividad para generar automáticamente tareas, descomponiendo problemas más grandes en subtareas interrelacionadas para una solución eficiente.

Justificación:
El proyecto busca proporcionar una comprensión práctica y teórica de estas estructuras de datos, resaltando su utilidad en la solución de problemas reales en la gestión de tareas. Además, busca establecer un marco que permita adaptar soluciones específicas según las necesidades del proyecto, mejorando la eficiencia y la capacidad de gestión en entornos cada vez más complejos.
