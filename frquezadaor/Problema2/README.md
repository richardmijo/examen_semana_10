# Sistema de Administración de Proyectos con Listas Circulares

## Propósito del Proyecto

Este proyecto implementa un sistema de administración de proyectos utilizando listas circulares. El objetivo es gestionar proyectos y tareas de forma eficiente.
## Cómo Funciona el Código

El código implementa dos clases principales:
- **Clase `Tarea`:** Representa cada tarea de un proyecto con atributos como nombre, descripción, estado y un enlace (`siguiente`) a la siguiente tarea en la lista circular.
  
- **Clase `Proyecto`:** Representa un proyecto, que contiene una lista circular de tareas. Los métodos permiten agregar tareas, eliminar tareas y recorrer las tareas cíclicamente.

- **Clase `SistemaProyectos`:** Administra los proyectos de forma global, permitiendo agregar proyectos y listar todos los proyectos.
