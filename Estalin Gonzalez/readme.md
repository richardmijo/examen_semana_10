# Sistema de Administración de Proyectos

Este proyecto implementa un sistema de administración de proyectos utilizando estructuras de datos avanzadas en Python. El sistema permite gestionar proyectos y tareas mediante el uso de listas enlazadas, listas circulares y listas doblemente enlazadas. Además, se incluyen funcionalidades como la creación automática de tareas utilizando recursividad.

## Funcionalidades

- **Administración de proyectos**: Puedes agregar proyectos y asociarles tareas.
- **Gestión de tareas**: Cada tarea tiene un nombre, una descripción y un estado (pendiente, en progreso, completada).
- **Listas enlazadas**: Los proyectos y las tareas se gestionan mediante listas enlazadas simples.
- **Listas circulares**: Las tareas dentro de un proyecto pueden organizarse en una lista circular para permitir la revisión cíclica.
- **Listas doblemente enlazadas**: Las tareas pueden ser recorridas en ambas direcciones, y puedes insertar o eliminar tareas desde cualquier punto de la lista.
- **Generación automática de tareas**: Utilizando recursividad, el sistema permite generar tareas automáticamente en un rango de IDs.
